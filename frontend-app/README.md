# Spearhead Frontend

Minimal React UI for the responses-only (`/v1`) Spearhead flow.

## Commands

Install:

```bash
npm install
```

Dev server:

```bash
npm run dev
```

Build:

```bash
npm run build
```

## Notes

- Production/local integrated run is served by FastAPI from `frontend-app/dist`.
- Preferred local startup from repo root is:
  - `./scripts/dev-one-click.sh`
